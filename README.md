# 麻将牌谱生成器 (Mahjong Score Sheet Generator)

将日本麻将（立直麻将）天凤牌谱数据自动排版，生成精美的 A4 牌谱 PDF。

## ✨ 功能特性

- **单局模式** — 导入多局 JSON，生成每局独立 PDF + 整合 PDF
- **半庄模式** — 粘贴整场半庄 JSON，自动拆分为各局
- **天凤 URL 直转** — 输入天凤牌谱链接，一键获取 JSON 并生成
- **自定义标题/玩家名** — GUI 中修改，覆盖 JSON 内数据
- **PDF 输出控制** — 自由选择生成单局 PDF / 整合 PDF / 两者都生成
- **麻将字体渲染** — 手牌/副露/舍牌以真实麻将字体展示
- **标注系统** — 立直、点炮、自摸等标记清晰显示

## 🚀 快速开始

### 方式一：直接运行（推荐）

1. 下载最新 Release 中的 `麻将牌谱生成器.exe`
2. 双击运行（首次启动稍慢，约 3-5 秒）
3. 开始使用！

### 方式二：Python 源码运行

**环境要求：**
- Python 3.10+
- Node.js 18+（天凤 URL 转换功能需要）

**安装步骤：**

```bash
# 1. 克隆仓库
git clone https://github.com/你的用户名/MahjongScoreSheetGenerator.git
cd MahjongScoreSheetGenerator

# 2. 一键安装（推荐）
setup.bat

# 或手动安装
pip install -r requirements.txt
cd tenhou/tenhou-convert && npm install && cd ../..

# 3. 运行
python batch_gui_form_v4.py
```

## 📖 使用说明

### 单局模式（默认）

1. 点击 **[+] 新增一局** 创建空白牌局卡片
2. 在 JSON 编辑区粘贴单局 JSON 数据
3. 绿色 ✓ 表示 JSON 格式正确，红色 ✗ 表示有误
4. 可添加多局（↑↓ 排序、删除）
5. 勾选底部 **生成单局 PDF** / **生成整合 PDF**
6. 点击 **生成整合牌谱 PDF**，选择输出目录

### 半庄模式

1. 切换到 **半庄模式**
2. 粘贴完整半庄 JSON（`log` 包含 8~12 局），或点击 **导入半庄 JSON...**
3. 点击 **解析预览** 查看每局摘要
4. 也可在 **天凤 URL** 输入框粘贴牌谱链接，点击 **获取**

### 天凤 URL 获取

支持以下格式：
- `https://tenhou.net/0/log/?2026052817gm-00a9-0000-e87d70de`
- `https://tenhou.net/0/?log=2026052817gm-00a9-0000-e87d70de&tw=3`
- 纯牌谱 ID：`2026052817gm-00a9-0000-e87d70de`

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + N` | 新增一局 |
| `Ctrl + O` | 导入 JSON 文件 |
| `Ctrl + G` | 生成 PDF |
| `Ctrl + S` | 保存项目 (.mjproj) |

## 📋 JSON 格式

### 单局 JSON

```json
{
  "title": ["牌局标题", "2024-01-01"],
  "name": ["Player1", "Player2", "Player3", "Player4"],
  "rule": { "disp": "电脳麻雀", "aka": 1 },
  "log": [
    [
      [0, 0, 0],
      [25000, 25000, 25000, 25000],
      [21],
      [18],
      [16, 17, 19, 21, 25, 27, 27, 28, 35, 45, 45, 46, 46],
      ...玩家2~4数据...,
      ["和了", [-5200, 0, 0, 7200], [3, 0, 3, "40符3飜5200点", ...]]
    ]
  ]
}
```

### 半庄 JSON

与单局格式相同，`log` 数组包含多局：

```json
{
  "title": ["半庄标题", "2024-01-01"],
  "name": [...],
  "rule": {...},
  "log": [
    [ /* 东1局 */ ],
    [ /* 东2局 */ ],
    ...
    [ /* 南4局 */ ]
  ]
}
```

## 🎨 标注符号

| 符号 | 含义 |
|------|------|
| R | 荣和 |
| M | 自摸 |
| r | 立直 |
| C | 放铳 |
| k | 杠 |
| m | 明杠 |
| a | 暗杠 |
| j | 加杠 |
| d | 摸切（分隔符） |
| N | 空牌 |
